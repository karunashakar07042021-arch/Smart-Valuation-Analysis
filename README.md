# Interactive Analytics for Smart House Price Valuation
###  Objective (Problem Statement)

The objective of this project is to analyze housing data and build a model that can accurately predict house prices based on various features such as location, size, number of rooms, and other property attributes. This helps in understanding the key factors influencing house prices and supports better decision-making in real estate.
### This project focuses on:
* Cleaning and preprocessing raw housing data
* Performing exploratory data analysis (EDA) to identify trends and patterns
* Identifying key factors affecting house prices (e.g., area, location, amenities)
* Developing interactive dashboards for data visualization using Power BI  and python
* Building a predictive model to estimate house prices
* Providing data-driven insights for smarter price prediction.

  ## **Outcome**
* 📊 Identified key factors influencing house prices such as area, condition, and year built
* 📈 Discovered important trends and patterns through data analysis and visualization
*  💡 Provided actionable insights to support better pricing and real estate decisions

  ## **Domain**
The **housing market** (real estate domain) involves the buying, selling, and renting of residential properties such as houses, apartments, and villas. Prices in this market are highly dynamic and influenced by multiple economic, geographic, and property-specific factors.

This domain plays a crucial role in:

* Economic growth
* Urban development
* Investment planning

#**Dataset Information**

**Source:**  Open Data Portals: <font color="blue"><u> **www.geeksforgeeks.org**</font></u>

**Year / Timeline:** Historical housing data (approx. early 2000s to recent years)

**Dataset Column/features Description:**

* **Id:** Unique identifier for each house.
* **MSSubClass:** Type of dwelling (building class).
* **MSZoning:** General zoning classification of the property.
* **LotArea:** Lot size in square feet.
* **LotConfig:** Lot configuration (corner, inside, etc.).
* **BldgType:** Type of building.
* **OverallCond:** Overall condition rating of the house.
* **YearBuilt:** Year the house was built.
* **YearRemodAdd:** Remodel year (same as construction year if no remodeling).
* **Exterior1st:** Exterior covering of the house.
* **BsmtFinSF2:** Type 2 finished square feet of the basement.
* **TotalBsmtSF:** Total basement area in square feet.
* **SalePrice:** The target variable we aim to predict.
