# Interactive Analytics for Smart House Price Valuation
###  Objective (Problem Statement)

The objective of this project is to analyze housing data and build a model that can accurately predict house prices based on various features such as location, size, number of rooms, and other property attributes. This helps in understanding the key factors influencing house prices and supports better decision-making in real estate.
### This project focuses on:
* Cleaning and preprocessing raw housing data
* Performing exploratory data analysis (EDA) to identify trends and patterns
* Identifying key factors affecting house prices (e.g., area, location, amenities)
* Developing interactive dashboards for data visualization using Power BI  and python
* Building a predictive model to estimate house prices
* Providing data-driven insights for smarter price prediction.

  # **Outcome**
* 📊 Identified key factors influencing house prices such as area, condition, and year built
* 📈 Discovered important trends and patterns through data analysis and visualization
*  💡 Provided actionable insights to support better pricing and real estate decisions
